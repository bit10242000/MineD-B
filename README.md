# MineD&E&S

### Database 
A minerals Database containing data like the mineral, mining company, processing factory, GPS location of the factory, map and location services for processing factories, purity , quality and quantity of mineral mined in the last 24 hours, photographs of processed mineral, inventory of mineral held by a processing factory at the present, price of mineral per weight, quantity of a mineral ore deposits and dashboard with a theme colour of a mineral for these data and a search engine for these data. The providers of these data are processing factories.
